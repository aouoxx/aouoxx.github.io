#### java char 转换为 int 类型

```java
char ch = '9';
if(Character.isDigit(ch)){
	int num = Integer.parseInt(String.valueOf(ch));
    sout(num);
}


char ch='9';
if(Character.isDigit(ch)){
    int num = (int)ch-0;
    sout(num);
}
```
